Introducción

Matlab es una herramienta computacional muy poderosa que se ocupa en muchas ramas de ingenieria y ciencia. El nombre Matlab proviene del hecho que usa un lenguaje basado en matices, pero el software no solo se limita a un lenguaje también se pueden desarrollar aplicaciónes gráficas, ejecutar análisis de datos, realizar aplicaciones web o de IOT, podemos optimizar código de c y c++, tiene compatibilidad con muchos hardwares como Arduino y Raspberry Pi, además de incluir una herramienta llamada Simulink. Simulink es un entorno de desarrollo de diagramas de bloques multidominio basado en diseño modelado. Las aplicaciones que se pueden desarrollar en matlab pueden ser de los siguientes campos:

  • Análisis de datos
  • Comunicaciónes Inalámbricas
  • Deep Learning
  • Visión Artificial
  • Procesamiento de señales
  • Finanzas Cuantitativosy Gestión de Riesgos
  • Robótica
  • Sistemas de Control
